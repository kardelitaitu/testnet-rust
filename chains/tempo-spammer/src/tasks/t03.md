task type: token transfer

detailed sequence of task 03:
1. randomly select one of the 4 system tokens (PathUSD, AlphaUSD, BetaUSD, ThetaUSD)
2. query the token balance using ERC20 balanceOf (selector: 0x70a08231)
3. check if balance >= 1,000,000 (1M units) - minimum threshold
4. if balance is sufficient, calculate 2% of balance for transfer
5. generate a random recipient address
6. construct transfer calldata (selector: 0xa9059cbb + 32 bytes recipient + 32 bytes amount)
7. send the transfer transaction
8. wait for confirmation and return tx hash
