# 09. Transfer Token

## Process
1. Load system tokens and created stablecoins from database.
2. Shuffle and check balance for up to 5 tokens.
3. Select first token with > 100,000 wei balance.
4. Generate random amount (10-50 units) or use 50% of balance if insufficient.
5. Generate random recipient address.
6. Determine Fee Token:
   - If System Token: Use itself.
   - If Created Token: Use random System Token.
7. Build `transfer(address,uint256)` calldata.
8. Send transaction and verify receipt.
