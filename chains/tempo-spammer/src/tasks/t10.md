# Task 10: Transfer with Memo

## Workflow
1. Select PathUSD token.
2. Check balance.
3. Generate a random memo (text + bytes32).
4. Send `transferWithMemo` transaction to a random address.

## Inputs
- Wallet with PathUSD balance.
- Random amount (10-50 units).
- Random memo.

## Outputs
- Transaction Hash.
- Log message with memo details.
