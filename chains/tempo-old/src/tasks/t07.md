# Task 07: Mint Stablecoin

## Workflow
1.  **Retrieve Assets**: Queries `tempo.db` (table `created_assets`) for stablecoins previously created by the current wallet.
2.  **Select Asset**: Randomly selects one of the retrieved stablecoins to mint.
3.  **Check Role**:
    *   Checks if the wallet has the `ISSUER_ROLE` on the selected token contract.
    *   If missing, it attempts to `grantRole` (this is possible if the wallet is the admin/owner).
4.  **Decimals Lookup**: Queries the token's `decimals()` to ensure accurate minting.
5.  **Mint**: Submits a `mint` transaction for a random amount between **10,000,000** and **20,000,000** tokens.
6.  **Gas Management**: Uses `GasManager` with a 5.0x multiplier to ensure the transaction is prioritized.

## Technical Details
*   **Implementation**: `chains/tempo/src/tasks/t07_mint_stable.rs`
*   **Role Identification**: `ISSUER_ROLE = keccak256("ISSUER_ROLE")`
*   **Target**: The selected wallet's own address.

## How to Debug
Run this specific task with a selected wallet:
```bash
cargo run --bin debug_task -- --task 7 --wallet 0
```

## Success Criteria
*   Transaction hash generated for `mint` call.
*   Confirmation of `ISSUER_ROLE` or successful grant.
*   Zero compilation warnings.
