# Task 05: Swap Stablecoin

## Workflow
This task executes a swap between two stablecoins (System or Created) on the Tempo Stablecoin DEX. It implements best practices for DeFi interactions, including price quoting and slippage protection.

### Steps
1.  **Selection**: Randomly selects two different tokens from the pool of System Tokens (`PathUSD`, `AlphaUSD`, etc.) or user-created tokens.
2.  **Quote**: Queries the DEX (`quoteSwapExactAmountIn`) to check liquidity and calculate the expected output for 3% of the wallet's balance.
3.  **Slippage**: Calculates a minimum output amount (`minAmountOut`) with a **20% slippage tolerance** (80% of quoted amount). This high tolerance helps handle volatile testnet conditions.
4.  **Approve**: Checks the token allowance for the DEX. If insufficient, sends an infinite approval (`type(uint256).max`).
5.  **Swap**: Calls `swapExactAmountIn` with the calculated parameters.
6.  **Verify**: Logs the transaction hash.

## Technical Details
*   **Implementation**: `chains/tempo/src/tasks/t05_swap_stable.rs`
*   **DEX Contract**: `0xdec0...`
*   **Gas Limit**: Hardcoded to 2,000,000 to prevent OOG errors during complex routing.
*   **Safety**: Returns success (but logs warning) if the quote is zero, handling liquidity shortages gracefully without crashing the bot.
